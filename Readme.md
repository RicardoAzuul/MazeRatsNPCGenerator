## Usage

MazeRatsNPCGenerator is a Command Line Interface application that parses a bunch of text files containing many of the NPC tables from Maze Rats, generates numbers between 1 and 6
and then outputs a log file containing NPCs with all the traits randomly generated in the text files.
If you run it without any arguments, it generates a default of 10 NPCs.
If you run it with /?, it displays a short help text.
If you run it with a number, it will generate that many NPCs.
